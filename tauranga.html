  1. To create a new live asset, click "Add new row

  2. Select the radio / GPS number from the first dropdown list. These are sorted into headings and by number order

  3. Enter the team name for the user or team that is going out with the asset

  4. Clicking on the claendar icon brings up a date and time picker, clicking "today" will automatically enter the current date and time.

  5. Finally, the black arrow will move the asset card into the out section. To return it, just click the arrow to move it back up

  6. Note: "Remove Row" will delete the row forever!
